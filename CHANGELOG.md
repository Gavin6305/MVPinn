# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-07-XX

### Added
- Initial release of PINN4ME
- Physics-Informed Neural Network for Milne-Eddington inversion
- Training pipeline with adaptive loss weighting
- Batch inference support
- FITS file data loading
- Example notebook demonstrating usage

### Features
- ME forward model implementation in PyTorch
- Voigt profile calculation
- Adaptive Stokes component weighting
- Support for both Adam and L-BFGS optimizers
- GPU acceleration support
